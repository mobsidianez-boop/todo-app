# Todo App

個人用のシンプルなTodoアプリです。

## Features

- タスクの追加
- 状態の管理（未着手 / 着手中 / 完了）
- 完了への変更
- タスク削除
- 状態ごとの絞り込み
- 完了済みタスクの件数と内容の確認
- 操作時のアニメーション
- `localStorage` によるブラウザ内保存

## GitHub Pages

`index.html` がリポジトリルートにあるため、GitHub Pages の Source を `Deploy from a branch`、Branch を `main`、Folder を `/ (root)` に設定すると公開できます。
